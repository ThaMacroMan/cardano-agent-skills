# Cardano Agent Skills (Claude Code / Codex / Cursor)

This repo contains a set of **small, focused Agent Skills** for Cardano development:
wallets, cardano-cli transactions, staking, Plutus script transactions, Aiken, MeshJS, and Conway/Plutus V3 notes.

## Install (via npx)

### Option A — Vercel `skills` CLI
```bash
npx skills add realdecimalist/cardano-agent-skills
```

### Option B — `add-skill`
```bash
npx add-skill realdecimalist/cardano-agent-skills -a claude-code
```

## What you get

Skills live under `skills/` and each subfolder is one skill with its own `SKILL.md` entrypoint.

> Tip: Keep these skills updated to match your installed `cardano-cli` version. CLI syntax is evolving (era-prefixed commands, defaults, etc.).
